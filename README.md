# Dual Arena Game

## Setup Instructions

### 1. Database
- Open MySQL
- Run:
  database/database.sql

### 2. Backend
cd backend
npm install
node server.js

### 3. Unity
- Open project in Unity Hub
- Press Play

## Features
- 1v1 battle system
- Tournament mode
- Player stats (win rate, moves)
- Backend API (Node.js + MySQL)

## Notes
- Make sure MySQL is running
- Update DB credentials in backend/db.js if needed
